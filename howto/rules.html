Entry rules

Aggregator uses entry rules to filter out the entries of a feed.

There are two types of rules: exclude rules, used to mark which entries should be hidden while aggregating a feed, and include rules, used to cancel the effect of some exclude rules.

Every entry rule defines a mask used to match the entries for which the rule action is applied. The % character is used to match zero, one or multiple characters, and can be used multiple times in a mask.

For example, the %aggregator% title mask will match every entry that contains aggregator in the title. While the % Vol % Ch % title mask will match every entry that contains both  Vol  and  Ch  (in the same order as present in the mask) in the title.

Excluding unwished entries

The easyest way to define rules that hide entries of a feed is by using the Exclude action:

  1. Select the entry which you want to exclude by long pressing it.
  2. Click the Exclude action from the action bar.

NOTE 1: Depending on your device type, this action can be found as icon or as option in the overflow menu.

NOTE 2: Long pressing an icon in the action bar reveals the action name.
